# RAG-Chatbot-Gen-AI
# RAG PDF + Website Chatbot

A GenAI-powered chatbot that answers questions from:
- Uploaded PDFs
- User-provided Website URLs

Tech Stack:
- Streamlit
- LangChain
- FAISS
- Google Gemini

Features:
- PDF Question Answering
- Website Question Answering
- Retrieval Augmented Generation (RAG)
- Semantic Search
